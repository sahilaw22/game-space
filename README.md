## Overview

**Full Throttle Offroad Truck** is a 2D physics-based driving challenge built around control, momentum, and split-second decisions. Every track pushes the player through steep climbs, unstable landings, broken paths, and high-commitment jumps where one error can end the run.

The game is designed to feel immediate for new players and demanding for skilled players. You can start quickly, but finishing clean runs requires rhythm, patience, and accurate truck balance.

## Core Features

- Real-time vehicle behavior driven by momentum and body balance
- Track layouts with moving platforms, narrow ledges, and jump chains
- Minimal controls with high mechanical depth
- Responsive design with touch support for phones and tablets
- Hand-crafted terrain flow designed around timing windows
- Short attempt cycles that keep every restart meaningful

---

## Controls

| Input | Action |
| --- | --- |
| `Arrow Keys` or `A / D` | Adjust truck balance left and right |
| `Space` or `W` | Accelerate or trigger jump |

### Driving Principles

- Keep the chassis level before and after each jump
- Control speed before cliff edges, not after
- Use momentum as a tool, not constant full throttle

---

## Objective

Clear each course without flipping your truck. Difficulty scales through terrain complexity, tighter landing zones, and more punishing gaps, turning every stage into a control test rather than a speed sprint.

---

## Level Design Language

Each level combines multiple hazard types to force adaptation during a run:

- Moving platform timing checks
- Narrow ledge alignment sections
- High-risk jump links
- Uneven traction zones
- Sharp vertical drops and recovery ramps
